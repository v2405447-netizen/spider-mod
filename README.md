# Spider Mod для Minecraft 1.16.5

Это мод позволяет играть за паука в Minecraft!

## Возможности

1. **Карабканье по стенам** - Автоматически карабкайтесь вверх по стенам со скоростью 0.2 блока/тик
2. **Хождение по воде** - Бегайте по поверхности воды как по земле
3. **Бесконечная установка паутины** - Используйте предмет "Паутина" для установки паутины
4. **Отсутствие замедления в паутине** - Паутина не замедляет паука
5. **Лазанье по паутине** - Карабкайтесь по своей паутине (Пробел вверх, Shift вниз)
6. **Ядовитый укус** - Удары рукой накладывают яд на цель
7. **Отсутствие урона от падения** - Падайте с любой высоты без урона
8. **Стрельба паутиной** - Используйте "Паучий лук" для стрельбы нитями
9. **Размножение пауков** - Shift + ПКМ по пауку для создания маленького паучка

## Установка

1. Убедитесь, что у вас установлены Fabric Loader и Fabric API для 1.16.5
2. Скомпилируйте мод: `gradlew build`
3. Найдите `spider-mod-1.0.0.jar` в папке `build/libs/`
4. Поместите JAR в папку `mods/` в вашем Minecraft

## Разработка

Мод использует:
- Minecraft 1.16.5
- Yarn Mappings 1.16.5+build.10
- Fabric Loader 0.11.0+
- Fabric API 0.42.0+

## Требования

- Java 8+
- Gradle
