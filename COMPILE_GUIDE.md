# 🕷️ Гайд по компиляции Spider Mod

## Быстрый способ (для ленивых)

### На Windows:
1. Открой папку с модом
2. Нажми `Shift + ПКМ` → "Открыть окно PowerShell здесь"
3. Выполни:
```bash
.\gradlew.bat build
```
4. Жди 2-3 минуты
5. JAR появится в `build/libs/spider-mod-1.0.0.jar`

### На Linux/Mac:
```bash
./gradlew build
```

## Что дальше?

1. Скопируй `spider-mod-1.0.0.jar`
2. Положи в папку `~/.minecraft/mods/` (или в папку mods твоего профиля в PrismLauncher)
3. **ВАЖНО:** В профиле должны быть установлены:
   - Fabric Loader 0.11.0+
   - Fabric API 0.42.0+
4. Запусти игру
5. Проверь меню "Моды" - Spider Mod должен быть там

## Если не работает

- Проверь, что установлена Java 8+
- Посмотри логи в `.minecraft/logs/`
- Скажи, какая ошибка выводится

## Требования

- Java 8 или выше
- Gradle (входит в проект)
- Интернет (для загрузки зависимостей)
